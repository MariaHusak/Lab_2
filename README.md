# Paw Runner

**Paw Runner** — це 2D-платформер на Python із використанням PySide6.  
Гравець керує собакою, яка стрибає через перешкоди. Рахунок підраховується за пройдений час, а гра завершується при зіткненні з перешкодою.

---

## Мета гри

- Управляти собакою та уникати перешкод.  
- Набрати максимальний рахунок (Score) без зіткнень.  
- Після завершення гри натиснути `R` для перезапуску.

---

## Особливості

- Реалізована фізика стрибка з гравітацією та утриманням клавіші для довшого стрибка.  
- Динамічна генерація перешкод з випадковою висотою.  
- Відображення рахунку в реальному часі.  
- Стильний простий 2D GUI на PySide6.
